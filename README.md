# SkillGap AI 🤖

**Know exactly where you stand — before you apply.**
       
Ever wondered why you're not getting calls even after applying to 50+ jobs?  
Most times it's a skill gap you didn't even know existed.  
SkillGap AI fixes that.

Enter your skills or upload your resume → get your readiness score, see exactly what's missing, and get a week-by-week AI learning roadmap — all in seconds.

---

## ✨ What It Does

- 🎯 **Readiness Score** — how ready you actually are for your target role (out of 100%)
- 🔍 **Skill Gap Analysis** — matched vs missing skills, ranked by how important they are
- 🗺️ **AI Learning Roadmap** — week-by-week plan generated by Llama 3 (not generic advice, actually tailored to YOUR gaps)
- 📄 **Resume Upload** — upload your PDF and let AI extract your skills automatically
- ✍️ **Manual Entry** — or just type your skills as a comma-separated list

---

## 🛠️ Built With

- **Python + Flask** — backend
- **Groq API (Llama 3.3 70B)** — roadmap generation + resume skill extraction
- **pdfplumber** — PDF text extraction
- **HTML + CSS + Vanilla JS** — frontend (dark theme, Space Grotesk + Inter)

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/thepratikshamishra/skillgap-ai.git
cd skillgap-ai
```

### 2. Create virtual environment
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Add your Groq API key
Create a `.env` file in the root folder:
```
GROQ_API_KEY=your_key_here
```
Get a free key at [console.groq.com](https://console.groq.com)

### 5. Run it
```bash
python app.py
```
Open `http://127.0.0.1:5000` and you're good 🎉

## 💡 How It Works

```
you enter skills / upload resume PDF
        ↓
PDF text extracted → Llama 3 pulls out your technical skills
        ↓
skills matched against role requirements (weighted)
        ↓
readiness score = matched weight ÷ total weight × 100
        ↓
critical gaps sorted by importance
        ↓
Llama 3 generates your personal week-by-week roadmap
        ↓
everything rendered on a clean dark UI ✨
```
