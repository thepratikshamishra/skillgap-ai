## Features

- **Readiness Score** — calculates how ready you are for your target role as a percentage
- **Skill Gap Analysis** — matched vs missing skills ranked by importance
- **AI Learning Roadmap** — week-by-week plan generated by Llama 3 via Groq API
- **Resume Upload** — upload your PDF resume and let AI extract your skills automatically
- **Manual Entry** — or just type your skills as a comma-separated list
- **5 Target Roles** — Data Analyst, Data Scientist, Web Developer, Python Developer, Machine Learning Engineer

## Setup

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/skillgap-ai.git
cd skillgap-ai
```

### 2. Create virtual environment
```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Mac/Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Add your Groq API key
Create a `.env` file in the project root:
```
GROQ_API_KEY=your_groq_api_key_here
```
Get your free API key at [console.groq.com](https://console.groq.com)

### 5. Run
```bash
python app.py
```
Open `http://127.0.0.1:5000`

## How It Works

```
User inputs skills / uploads resume PDF
        ↓
PDF text extracted → Llama 3 pulls out technical skills
        ↓
Skills matched against role requirements in roles_skills.json
        ↓
Readiness score calculated (weighted match ÷ total weight × 100)
        ↓
Critical gaps identified and sorted by importance
        ↓
Llama 3 generates a week-by-week learning roadmap
        ↓
Results rendered on a clean dark-themed UI
```
